# Rust Raid Base Images

A collection of raid base images for the Rust Raid Selector plugin, organized by creator.

## Folder Structure

Images should be organized using the following structure:

```
creator/
└── base_pack_name/
    ├── image1.jpg
    ├── image2.jpg
    └── image3.jpg
```

**Example:**
```
trapbox/
└── santa_bases/
    ├── santa1.jpg
    ├── santa2.jpg
    └── santa3.jpg
```

## Image Requirements

- **Format:** Images should be in `.jpg` or `.png` format. **JPG is preferred** for smaller file sizes.
- **Naming:** Use descriptive, lowercase filenames without spaces (use hyphens or underscores instead).

## Contributing

To add new images:

1. Fork this repository
2. Add your images following the folder structure above
3. Create a Pull Request with a description of the bases being added
4. Include the creator name and base pack name in your PR description

All contributions are welcome!
